# Community LiveKit Dashboard

An open-source alternative to LiveKit Cloud Dashboard, built using LiveKit's open-source components (Server, SDKs, SIP, Agents).

## Overview

This project provides a community-driven dashboard for managing LiveKit rooms, agents, and SIP integrations without subscription costs. It enables full self-hosting and customization.

## Features

- Room management (create, monitor, control)
- Agent orchestration (deploy, manage AI agents)
- SIP integration for voice agents
- Real-time monitoring and analytics
- Customizable UI and API

## Tech Stack

- Frontend: React
- Backend: Node.js
- Integration: LiveKit Server, WebRTC

## Setup

1. Clone the repo
2. Install dependencies for frontend and backend
3. Run LiveKit Server
4. Start the dashboard
